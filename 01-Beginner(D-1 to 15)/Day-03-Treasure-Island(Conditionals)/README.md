# 🏝️ Day 03 — Treasure Island

## 📌 Project Overview

Treasure Island is a text-based adventure game where the player makes decisions to navigate through different scenarios in search of hidden treasure. Each choice determines the outcome of the game.

---

## 🎯 Learning Objectives

- Understand conditional statements
- Practice nested `if` and `elif` statements
- Build logical decision-making skills
- Create an interactive command-line game

---

## 🧠 Concepts Learned

- `if`, `elif`, and `else`
- Nested Conditional Statements
- Logical Thinking
- User Input
- String Comparison
- Program Flow

---

## 📂 Files

- `treasure_island.py`

# 💰 Day 02 — Tip Calculator

## 📌 Project Overview

The Tip Calculator is a beginner Python project that calculates how much each person should pay after splitting a bill, including a chosen tip percentage.

---

## 🎯 Learning Objectives

- Work with variables and user input
- Perform mathematical calculations
- Convert between data types
- Format decimal numbers
- Display user-friendly output

---

## 🧠 Concepts Learned

- Variables
- Data Types (`int`, `float`, `str`)
- Type Conversion
- Mathematical Operations
- User Input
- f-Strings
- Number Formatting

---

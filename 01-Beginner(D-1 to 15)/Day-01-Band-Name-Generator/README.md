# 🎸 Day 01 — Band Name Generator

 📌 Project Overview

The Band Name Generator is a beginner Python project that generates a creative band name using the user's hometown and pet name.

---

# 🎯 Learning Objectives

- Understand Python syntax
- Learn how to use variables
- Take user input
- Combine strings to generate output

# 🧠 Concepts Learned

- input()
- Variables
- String Concatenation
- Python Basics
